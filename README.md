# Ristinolla

Ristinolla on peli jossa pelataan vastakkain. Kyseisessä variaatiossa voidaan luoda pelialusta mielivaltaisella koolla. On kuitenkin pieniä rajoituksia. koko maksimissaan 200 ja minimissään 2 ruutua.
